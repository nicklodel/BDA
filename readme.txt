Esta es la aplicación web realizada para la asignatura de BBDDAA. Para ejecutarla hacen falta las siguientes tecnologías:
react
express
nodejs

Se trata de un programa que, mediante una API REST que pide datos a la base de datos alojada en oraclepr.uco.es,
realiza consultas de vista,inserción,modificación y eliminación.
Para ejecutarlo, los pasos a seguir son:
1. Conectarse a la VPN de la UCO.
2.Ejecutar el backend en su carpeta con npm start
3.Ejecutar el frontend en su carpeta con npm start.

Con esto, se despliegará una vista de la base de datos con todo lo necesario para realizar las consultas mencionadas
